# zundamon-explainer-samples

YouTube「**ずんだもんのAI開発レベルアップ日記**」で紹介した仕組みを、手元で動かせるサンプルにしたものです。

各回の「AIに何を作らせたか／どう直したか」を、**最小・匿名のコード**で再現しています。実プロダクトのコードではなく、**概念を学ぶための教材**として書き直しています。

## 前提

- **Claude Code で開くこと前提**です。各フォルダの `README.md` に「何を見るか」「どう動かすか」「Claude Code にどう頼むか」を書いています。
- Python 3.10+ を想定（特別な記載がなければ追加インストールなしで動きます）。

## サンプル一覧

| テーマ | フォルダ / リンク | ひとことで |
|--------|------------------|-----------|
| テストが本物のLLMを呼んでいた | [`no-llm-in-tests/`](./no-llm-in-tests) → [OSS実物へ](https://github.com/todoroki-godai/no-llm-in-tests) | テストからLLM呼び出しを締め出す2層ガード |
| 点数を決めたらAIが自分でプロンプトを直す | [`eval-first-prompt/`](./eval-first-prompt) | 「直し方」より先に「点数の付け方」を決める |
| 稼働ゼロなのに毎月課金される“亡霊コスト” | [`ghost-cost/`](./ghost-cost) | 増え方を見張り、新顔の課金を見つける番犬 |
| AIに任せたらブラウザを40個開いて暴走 | [`ai-runaway-browser/`](./ai-runaway-browser) | 暴れない“作法”＋プロセス見張り＋自分の分だけ片付け |

> フォルダはテーマ名で並べています。動画の本数や公開順は変わることがあります。
