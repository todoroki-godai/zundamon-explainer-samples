# “1枚ずつ”をやめて“まとめて”片づける

動画の要点：たくさんの書類（領収書のような書類）をAIに“丸投げ”して「1枚ずつ目で見て」仕分けさせると、
重い処理が件数ぶん積み上がり、**AIが一度に扱える量（コンテキストの上限）を超えて途中で止まる**。
AIエージェントを何台も並行で増やしても、各自が同じ壁にぶつかるので解決しない。
効くのは“台数”ではなく“やり方”＝**キーワードでざっくり仕分け → 文字で読める書類はそのまま抽出 →
画像の書類だけ目で見る → 自信がないものだけ人が確認**、という一括処理への作り替え。

このサンプルは、その違いを**AIを呼ばずに**「処理回数」と「概算トークン」のそろばんで確かめる教材です。

## このサンプルが示すこと

- `compare.py` … ダミーの書類を大量に作り、**ループ版**（全部を1枚ずつ目で見る）と
  **一括版**（仕分け→文字抽出→画像だけ目で見る）で、重い処理の回数と概算トークンを比べます。
  標準ライブラリのみ・**本物のAI/LLMは一切呼びません**（APIキー不要）。

> **注記**: 数字はすべて分かりやすいダミーです（画像1枚≒1,500トークン等）。実プロダクトの実測値ではなく、
> 「1件ずつ積み上げると上限を超える／まとめると重い処理がごく僅かになる」という“形”を見るためのものです。

## まず動かす

```
python compare.py
```

（macOS など `python` が見つからない場合は `python3 compare.py` でも同じです）

実際の出力はこちら（実機で走らせた結果をそのままコピペしています）：

```
書類 1,300 件（うち画像 186 件／電子 1,114 件）
AIが一度に扱える量の目安（コンテキスト上限・ダミー）: 200,000 トークン
============================================================

【ループ版】全部を“1枚ずつ目で見て”読む
  必要な“重い読み取り”: 1,300 回
  それに必要な概算トークン: 1,950,000（上限の約 9 倍）
  → 133 件目あたりで上限に達して ❌ 停止（残りは手つかず）

【一括版】キーワード仕分け → 電子は文字抽出 → 画像だけ目で見る
  文字でそのまま読めた（軽い）: 1,114 件
  “目で見た”（重い）: 186 件だけ
  1件あたりの最大トークン: 1,500（独立処理なので積み上がらない → 上限に達しない）
  → 全 1,300 件、✅ 最後まで完走

============================================================
重い処理は 1,300 回 → 186 回（約 86% 削減）
台数を増やしたのではなく、“やり方”を変えただけ。
```

件数を変えると壊れ方が変わります：

```
python compare.py --count 500
```

## Claude Code での頼み方

自分のプロジェクトで「大量のくり返し」をAIに頼むときは、いきなり件数を渡す前に、こう相談してみてください。

```
このプロジェクトで、大量の同じ処理（例：たくさんのファイル/書類/レコードを1件ずつ処理する）を
AIに頼もうとしている。ループで1件ずつ回すのではなく、まとめて（一括で）処理できる方法や、
すでにある専用のプログラムがないか、一緒に探してほしい。
```

止まってしまったときは、数を増やす前にこう聞くのが近道です。

```
処理が途中で止まった。原因は“数（件数・台数）”か、それとも“やり方（1件ずつ読む重い処理）”か、
切り分けて教えて。やり方が原因なら、一括処理への作り替え案も出して。
```

## なぜ効くか・限界

**なぜ効くか**: 重いのは“1枚ずつ目で見る（画像として読む）”処理です。文字がそのまま取れる書類は
軽い抽出で済ませ、画像の書類だけを重い処理に回せば、重い処理の総数が一桁小さくなります。
さらに1件ずつ独立して処理すれば、1つの文脈に積み上がらないので上限に達しません。

**限界**:
- ここでの数字はすべてダミーの概算です。実際のトークン数やコストは、モデル・画像サイズ・
  文書の中身によって変わります（自分のデータで測ってください）。
- 「文字が取れる書類」と「画像の書類」の割合で効果は変わります（画像だらけなら削減幅は小さくなります）。
- 一括処理でも、最後に“自信がないものだけ人が確認する”一手間は残ります（そこは省かない）。

（`compare.py` の書類・分類・数値はすべて教材用のダミーです。自分の対象データに置き換えて使ってください。）
