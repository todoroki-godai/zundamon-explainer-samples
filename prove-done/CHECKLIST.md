# AIが「完了しました」と言ったら——目で確かめるチェックリスト

コードを書かない人向け。AIに作業を任せて「できました！」と返ってきたら、信じる前に次を確かめます。
**1つでも × なら“未完了”** として突き返します。

- [ ] **動かした？** 実際に起動・実行して、期待どおりの結果が出たか（出力やスクショで確認）
- [ ] **頼んだこと全部？** 依頼項目を1つずつ照合（「引き算も」と頼んだなら引き算も動くか）
- [ ] **“フリ”ワードが残ってない？** 「TODO」「とりあえず」「あとで」「仮実装」「ダミー」が成果物に残っていないか
- [ ] **テストは緑？** テストがあるなら全部通っているか（落ちたまま「完了」は詐欺）
- [ ] **証拠はある？** 「やりました」だけでなく、結果の証拠（ログ・出力・差分）を出させたか

## コツ：完了の定義を“先に”決める

作業を始める前に「何ができたら完了か」を箇条書きで書き、AIにも渡しておきます。
後出しにすると、AIの「完了」に合わせて基準がずるずる緩み、詐欺が通ります。
