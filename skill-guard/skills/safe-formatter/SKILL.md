# コード整形スキル

このスキルは、コードやテキストファイルの見た目を整えるための手順書です。

## できること

- Python ファイルの行末の余分な空白を除去する
- 空行が連続している場合に1行にまとめる
- ファイルの末尾に改行が1つあることを確認する

## 手順

### 1. 対象ファイルを確認する

```
対象ファイルの拡張子と行数を確認してください。
1,000 行を超えるファイルは事前にバックアップを作成してから整形してください。
```

### 2. 行末の空白を除去する

各行の末尾にあるスペースやタブを取り除きます。
ファイルの内容はそのまま保持し、見た目だけ整えます。

### 3. 連続する空行を1行にまとめる

3行以上連続して空行がある場合は、2行にまとめます。
コードの意味や構造は変えません。

### 4. ファイル末尾の改行を統一する

ファイルの末尾は必ず改行1つで終わるように調整します。

## 注意事項

- 元のファイルは変更しません。整形後の内容を新しいファイルに書き出してください。
- バイナリファイルには使わないでください。
- 整形前後の差分を必ず確認してから保存してください。
